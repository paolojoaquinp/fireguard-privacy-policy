POLÍTICA DE PRIVACIDAD

Alerta Incendios Bolivia

Última actualización: 1 junio 2025

1. INFORMACIÓN GENERAL

Esta Política de Privacidad describe cómo la aplicación "Alerta Incendios Bolivia" recopila, usa y protege la información personal de los usuarios. Nuestra aplicación está diseñada para facilitar el reporte y seguimiento de incidentes de incendios forestales en Bolivia.

2. INFORMACIÓN QUE RECOPILAMOS

2.1 Datos de Ubicación

  • Coordenadas GPS precisas cuando reportas un incidente de incendio
  • Ubicación actual para mostrar tu posición en el mapa
  • Datos de ubicación para geocodificación reversa (mostrar direcciones legibles)

2.2 Fotografías

  • Imágenes tomadas con la cámara del dispositivo para documentar incidentes
  • Fotografías seleccionadas de la galería para reportes de incidentes

2.3 Información de Usuario

  • Dirección de correo electrónico para autenticación
  • Nombre de usuario para identificación en reportes
  • Token de notificaciones push para alertas de incidentes

3. CÓMO USAMOS LA INFORMACIÓN

3.1 Propósito Principal

  • Reportes de Incidentes: Las fotografías y datos de ubicación se utilizan exclusivamente para documentar y reportar incidentes de incendios forestales
  • Mapas Interactivos: Los datos de ubicación permiten mostrar incidentes en mapas para facilitar la respuesta de emergencia
  • Notificaciones: Enviamos alertas sobre actualizaciones de incidentes cercanos a tu ubicación

3.2 Funcionalidades Específicas

  • Georreferenciación precisa de reportes de incendios
  • Visualización de incidentes en mapas de Mapbox
  • Seguimiento de actualizaciones de incidentes reportados
  • Mejora de la respuesta de emergencia mediante datos precisos

4. COMPARTIR INFORMACIÓN

4.1 No Vendemos Datos

No vendemos, alquilamos ni comercializamos tu información personal a terceros.

4.2 Servicios Utilizados

  • Firebase (Google): Para autenticación, almacenamiento de datos y notificaciones push
  • Mapbox: Para servicios de mapas y geocodificación
  • Autoridades de Emergencia: Los reportes de incidentes pueden ser compartidos con bomberos y autoridades competentes para respuesta de emergencia

5. ALMACENAMIENTO Y SEGURIDAD

5.1 Almacenamiento

  • Los datos se almacenan de forma segura en Firebase Cloud Firestore
  • Las fotografías se almacenan en Firebase Storage con acceso controlado
  • Los datos de ubicación se almacenan únicamente para el contexto del reporte

5.2 Seguridad

  • Utilizamos encriptación en tránsito y en reposo
  • Acceso autenticado mediante Firebase Auth
  • Cumplimos con las mejores prácticas de seguridad de Google Cloud

6. PERMISOS DE LA APLICACIÓN

6.1 Permisos Requeridos

  • Cámara: Para tomar fotografías de incidentes de incendios
  • Ubicación (Primer Plano): Para obtener coordenadas GPS precisas al reportar incidentes
  • Almacenamiento: Para acceder a fotografías de la galería
  • Internet: Para sincronizar datos y mostrar mapas

6.2 Permisos NO Utilizados

  • No accedemos a ubicación en segundo plano
  • No accedemos a contactos, mensajes o llamadas
  • No accedemos a micrófono para grabaciones de audio

7. RETENCIÓN DE DATOS

  • Los reportes de incidentes se mantienen para fines de registro histórico y análisis de patrones
  • Las fotografías se conservan mientras el incidente esté activo
  • Los datos de usuario se mantienen mientras la cuenta esté activa
  • Puedes solicitar la eliminación de tus datos contactándonos

8. TUS DERECHOS

  • Acceso: Puedes solicitar una copia de tus datos personales
  • Corrección: Puedes solicitar la corrección de datos inexactos
  • Eliminación: Puedes solicitar la eliminación de tu cuenta y datos
  • Portabilidad: Puedes solicitar tus datos en formato exportable

9. MENORES DE EDAD

Esta aplicación no está dirigida a menores de 13 años. No recopilamos intencionalmente información personal de menores de 13 años. Si descubrimos que hemos recopilado información de un menor, la eliminaremos inmediatamente.

10. CAMBIOS A ESTA POLÍTICA

Podemos actualizar esta Política de Privacidad ocasionalmente. Te notificaremos sobre cambios significativos mediante la aplicación o por correo electrónico. El uso continuado de la aplicación después de los cambios constituye tu aceptación de la nueva política.
